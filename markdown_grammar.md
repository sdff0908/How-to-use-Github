# 마크다운 문법

## 제목(heading)

제목은 `#`을 통해 표현한다. 제목의 레벨은 h1~h6까지 표현 가능하다.

## 목록

tab을 누르면 하위레벨에서 작성

shift+tab 누르면 상위 레벨로 올라옴

## 코드블록

```python
print('hi')
#python
```

```html
print(hi)
<!--HTML -->
```

```bash
print(hi)
#bash
```
